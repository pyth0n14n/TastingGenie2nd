# 利酒魔人2nd 仕様概要

## 1. 背景

本仕様は、2018年開発の「利酒魔人」から仕様のみを抽出し、
Androidアプリとして再構成したものである。

旧実装は使用せず、新規設計とする。

---

## 2. 目的

### ターゲット
- 日本酒をより深く理解したいユーザ
- 味覚・嗅覚・表現力を鍛えたいユーザ

### アプリ目的
- 利酒師の評価軸（色・香り・味）を選択式で記録する
- 同一銘柄のレビューを複数登録できる
- 後から振り返り可能にする

---

## 3. スコープ

### v1.0 に含む

- 酒（static）管理
- レビュー（dynamic）管理
- 画像添付
- ヘルプ表示
- ローカルDB保存
- インポート／エクスポート

### 将来拡張

- Web評価連携
- 嗜好分析・推薦
- 検索UI強化
- 利き酒モード
