# 画面遷移

## 画面一覧

| ID | 画面名 | 主な役割 | 遷移 |
|----|--------|----------|--- |
| S-1 | 初回オンボーディング / アプリの使い方 | 初回起動時に、酒登録・レビュー追加・選択式記録というアプリの基本操作を3ページの全画面スライドで説明する。設定の「アプリの使い方」からも再表示できる | S-1 → S1（最終ページの「酒を登録する」）<br> S-1 → S0（スキップ）<br> S-1 → S7（設定から再表示した場合のスキップ） |
| S0 | トップメニュー | 酒一覧表示、検索、設定導線 |S0 → S1（酒追加/編集）<br> S0 → S2（酒選択）<br> S0 → S5（画像選択）<br> S0 → S7（設定）<br> S0 → S0（酒削除確認/削除完了）|
| S1 | 酒登録/編集 | static情報入力 | S1 → S2 (追加完了) <br> S1 → S0 (キャンセル) <br> S1 → S1（入力後の戻る時は下書き破棄確認）|
| S2 | レビュー一覧 | レビュー参照 | S2 → S5 (画像選択) <br> S2 → S3（レビュー追加）<br> S2 → S4（レビュー詳細）<br> S2 → S2（レビュー削除確認/削除完了）|
| S3 | レビューの登録 | commonタイトル + 基本情報/外観/香り/味/特記タブ入力。新規レビュー入力の初回のみ、色を見る・香りを嗅ぐ・味わう・記録する流れを説明する全画面利き酒ガイドを表示する。右上の `?` から新規/編集どちらでも再表示できる | S3 → S2 (追加完了) <br> S3 → S2 (キャンセル) <br> S3 → S3（入力後の戻る時は下書き破棄確認）<br> S3 → S3（利き酒ガイド表示/閉じる） |
| S4 | レビュー詳細 | commonタイトル + summary + 香り・味のサマリ + accordion詳細、右上編集 | S4 → S3（編集）|
| S5 | 画像ビューア | 画像拡大表示 | S5 → S0/S2 (戻る) |
| S6 | 用語集 | 日本酒のきほん・用語説明 | Any |
| S7 | 設定 | 表示・操作、酒リスト・レビュー、データ、その他の静的メニュー表示。「アプリの使い方」からオンボーディングを再表示する。「このアプリについて」から20歳未満の飲酒禁止を明示する | S7 → S-1（アプリの使い方） |

## 初回起動

- `AppSettings.onboardingCompleted = false` の場合、起動先は S-1 とする。
- S-1 完了後は `onboardingCompleted = true` を保存し、最終CTAでは S1、スキップでは S0 へ遷移する。
- S-1 の1ページ目でのシステム戻るは無効化し、2ページ目以降では前ページへ戻る。
- 通常画面へ入った後、酒一覧または酒レビュー一覧が空の場合のみ、FAB付近に1回限りの短い吹き出しを表示する。中央CTAボタンは追加しない。

## レビュー入力ガイド

- `AppSettings.hasSeenTastingGuide = false` かつ S3 が新規レビュー入力の場合、S3 上に全画面の利き酒ガイドを自動表示する。既存レビュー編集では自動表示しない。
- 利き酒ガイドをスキップ、完了、システム戻るのいずれかで閉じた場合、`hasSeenTastingGuide = true` を保存する。
- S3 右上の `?` は `hasSeenTastingGuide` の値に関係なく利き酒ガイドを表示する。ガイド表示中のシステム戻るは S3 自体を閉じず、ガイドだけを閉じる。
