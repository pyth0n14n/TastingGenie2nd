# レビュー項目ヘルプメッセージ

| itemId | ヘルプメッセージ |
|---|---|
| APPEARANCE_SOUNDNESS | 透明性や清澄性を示す。異常な濁りや澱が無いかを確認。 |
| APPEARANCE_VISCOSITY | グラスの内側を流れ落ちる「脚」を見て判断する。アルコール度数と糖分が高いほど高くなる。日本酒は、基本的にはアルコール度数が15%前後で変わらないため、中程度に位置づけられる場合が多い。 |
| AROMA_UWATACHIKA | 飲む前に鼻で感じる香り示す。|
| AROMA_EXAMPLES | 何の要素（甘味、酸味、苦み、旨味）を思わせる何の香りかを示す。 |
| AROMA_MAIN_NOTE | 具体例と連動する香りの印象を形容詞などを用いて箇条書きで記す。 |
| AROMA_COMPLEXITY | 香り全体を加味し、香りの要素が多いか少ないかを示す。 |
| TASTE_ATTACK | 口に含んだ瞬間の印象を示す。時間軸で最初の印象。 |
| TASTE_TEXTURE | 感触などの組織構造を示す。食品では歯ごたえや舌触り、飲料では飲み口など。スパークリング日本酒の場合は泡たちのキメの細かさと捉える。 |
| TASTE_TEXTURE_NOTE | テイスティングノート モードの時のため。 |
| TASTE_SWEETNESS | 濃淳さ、ボリューム感を与える。テクスチャーに対して、なめらかさやまろやかさを与える。 |
| TASTE_SOURNESS | コクや厚みに影響。すっきりと爽やかな印象と、テクスチャにシャープさやキレの良さを与える。 |
| TASTE_UMAMI | 旨味が強いほど濃淳、少ないほどスッキリ、端麗となる。 |
| TASTE_BITTERNESS | テクスチャにシャープさやキレの良さを与えるとともに、辛味を感じる要因にもなる。 |
| TASTE_DESCRIPTION | 甘味、酸味、苦み、旨味などの中で、特筆すべき要素について言及する。 |
| TASTE_SWEET_DRYNESS | 甘さから辛さの程度を示す。個人差がある点に留意すること。アルコール・酸味・苦み・旨味など、複数要素が影響して単純な糖度だけで判断することが難しいため。 |
| TASTE_IN_PALATE_AROMA_INTENSITY | 口に含んだ状態から鼻に抜ける香りを示す。 |
| TASTE_AFTERTASTE_LENGTH | 最後に残る味わいの要素（キレ）を示す。時間軸で最後の印象。 |
| TASTE_AFTERTASTE_NOTE | 甘味、酸味、苦み、旨味などの中で、長さの長短を示す。 |
| TASTE_COMPLEXITY | アタック～余韻までの要素を加味し、味わいの要素が多いか少ないかを示す。 |
| OTHER_INDIVIDUALITY | 見た目、香り、テクスチャ、味わいから2～3要素を抽出して優先順位の高い要素から記載する。 |
| OTHER_CAUTIONS | 個性の中で、サービス/セールス時に留意すべき事項があれば記載する。 |
| OTHER_SAKE_TYPES | 味の複雑性 × 香りの強度 で4つに分類される。 |
| OTHER_FREE_COMMENT | 記載項目のない特記事項をメモする。 |
| OTHER_OVERALL_REVIEW | 3点を普通としたときの相対評価で記載する。 |
