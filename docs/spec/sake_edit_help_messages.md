# 酒登録ヘルプメッセージ

| itemId | ヘルプメッセージ |
|---|---|
| 麹米| 麹づくりに使用される米。 |
| 掛米| 醪づくりに使用される米。麹に水とともに加える。 |
| 精米歩合 | 玄米に対して、精米されて残った米の割合。精米歩合70%ならば、外側30%を削り70%が残っていることを意味する。|
| 日本酒度 | 水の比重を0とした時の日本酒の比重。一般に、マイナスほど甘く、プラスほど辛い傾向がある。 |
| 酸度 | 酒に含まれる有機酸（乳酸、リンゴ酸）の量を相対的に表す数値。一般に、高いほど辛く濃く、低いほど甘く淡く感じる傾向がある。|
| アミノ酸度 | 酒のコクや旨味のもとになるアミノ酸の量を相対的に表す数値。高いほど濃淳に、低いほど端麗に感じる傾向がある。|
| アルコール度数 | 飲料に含まれるアルコールの割合を、百分率（%）で表したもの。|
| 酵母 | 発酵によって糖質からアルコールや香りを生み出す微生物。種類により味や香りの特徴が変わる。|
| 水 | 仕込みに使う水。発酵や酒の品質・味わいに大きく影響する。|
