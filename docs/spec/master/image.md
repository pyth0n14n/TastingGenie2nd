# 画像仕様

## 1. 保持単位

- 画像は `Sake` に紐付ける
- 1酒につき 1件
- 保存項目は `imageUri: String?`

---

## 2. 取り扱い

- URIでのみ管理し、直接ファイルパスを前提にしない
- 画像が未設定でも酒保存を許可する
- 登録元は酒編集画面とする
- 酒編集画面では画像 preview を表示する
- 酒一覧画面では 2列カードで画像 preview を表示してよい
- 酒一覧画面の画像 preview は `setting_image_preview` が有効なときだけ表示する
- 酒一覧画面で画像未登録の酒は placeholder を表示する
- 画像削除は確認ダイアログを経由させる
- 画像の追加・差し替え・削除は保存時に確定させる
- 永続化時はアプリ専用領域へコピーし、永続 URI として扱う
- 画像表示は S5（画像ビューア）で行う
