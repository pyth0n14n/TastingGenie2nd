# 香りマスタ

## 1. 選択方式
- カテゴリごとに複数選択可
- 任意入力
- 保存形式：List<Enum>
- 対象項目は `aromaExamples` と `tasteInPalateAroma`
- `aromaExamples` は香りタブの具体例、`tasteInPalateAroma` は味わいタブの含み香として使う
- 旧 `scentBase` は使用しない
- 選択UIはステータスバー下からの全画面表示とし、スワイプ、画面外タップ、戻る操作では閉じない
- 閉じる操作は左上の閉じるボタン、下部左のキャンセルボタン、下部右の完了ボタンのみとする

## 3. メタ情報
| 香気成分 | 説明 | 相応しい形容詞例 |
|---|---|---|
| 吟醸香 | 酵母由来。主に薫酒に見られる要素 | 華やか、フルーティー、爽やか、清涼、みずみずしい |
| 原料香 | 原料由来。主に醇酒に見られる要素 | ふくよか、ふくらみ、まろやか、なめらか |
| 熟成香 | 熟成由来。主に熟酒に見られる要素 | 力強い、複雑、個性的、凝縮感、ボリューム感 |
