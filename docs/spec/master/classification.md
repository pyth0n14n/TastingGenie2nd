# 分類マスタ

## 1. 選択方式
- 複数選択（Checkbox）
- 任意
- OTHER 選択時は自由記述欄表示
- 「未選択」は選択肢に含めず、未設定の表示状態として扱う
- 解除は「クリア」で全解除する
- 検索付きの全画面 bottom sheet でカテゴリごとに折り畳み表示し、複数選択は「完了」で確定する
- OTHER を解除した場合、自由記述欄の値は保存対象から外し、UI上も空に戻す
