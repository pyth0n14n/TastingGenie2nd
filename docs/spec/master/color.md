# 色マスタ

## 選択方式
- 単一選択（Radio）
- 任意
- 対象項目は `appearanceColor`
- `OTHER` を選択した場合は `appearanceColorOther` に自由記述を保存する
- 選択済みの色はクリアできる
- レビュー編集では、温度選択と同じくラベル行右端にクリアを置き、下部シートで色を選択する
- 下部シートの選択肢は、表示名、色付きの `Water Drop` アイコン、ラジオボタンを並べる

| value | 表示名 |
|--------|--------|
| CLEAR | 無色透明 |
| PALE_YELLOW | 淡い黄色 |
| YELLOW | 黄色 |
| DARK_YELLOW | 濃い黄色 |
| BROWN | 茶色 |
| DARK_BROWN | こげ茶色 |
| GREEN | 緑 |
| OTHER | その他 |
