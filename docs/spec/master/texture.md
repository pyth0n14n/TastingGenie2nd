# テクスチャーマスタ

## 対象項目
- `tasteTextureRoundness`
- `tasteTextureSmoothness`

## 選択方式
- 2軸プロット UI
- 任意
- 未設定時は中央 placeholder を表示する
- タップまたはドラッグで X/Y の両軸を更新する

## X軸: 丸さ

| value | 表示名 |
|--------|--------|
| FIRM_TIGHT | シャープ |
| SLIGHTLY_FIRM_SHARP | ややシャープ |
| BALANCED | 中間 |
| SLIGHTLY_SOFT_ROUND | やや柔らかい |
| SOFT | 柔らかい |

## Y軸: 舌ざわり

| value | 表示名 |
|--------|--------|
| ROUGH | 粗い |
| SLIGHTLY_ROUGH | やや粗い |
| BALANCED | 中間 |
| SLIGHTLY_FINE | ややなめらか |
| SMOOTH | なめらか |
