# 温度マスタ

## 選択方式
- タップで Material Design 3 の modal bottom sheet を開き、一覧から選択する
- bottom sheet 背面は scrim で暗くする
- 一覧は温度ごとの色付きアイコン、表示名、目安温度、ラジオマークを表示する
- 任意
- 対象項目は `temperature`

| value | 表示名 | 目安温度 |
|--------|--------|----------|
| TOBIKIRI_KAN | 飛び切り燗 | 55℃以上 |
| ATSUKAN | 熱燗 | 50℃ |
| JOKAN | 上燗 | 45℃ |
| NURUKAN | ぬる燗 | 40℃ |
| HITOHADA | 人肌燗 | 35℃ |
| HINATA | 日向燗 | 30℃ |
| JOON | 常温 | 20℃ |
| SUZUHIE | 涼冷え | 15℃ |
| HANABIE | 花冷え | 10℃ |
| YUKIBIE | 雪冷え | 5℃ |
