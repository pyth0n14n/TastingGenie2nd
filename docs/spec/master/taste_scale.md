# 味わい評価マスタ

## 対象項目
- `tasteSweetness`
- `tasteSourness`
- `tasteBitterness`
- `tasteUmami`
- `tasteAftertaste`

## 選択方式
- 各項目ごとに5段階の段階選択 UI
- 任意
- 共通5段階を利用
- `tasteAftertaste` は同じ value を使うが、表示ラベルは「短い」〜「長い」の余韻向け表現にする

| value | 表示名 |
|--------|--------|
| VERY_WEAK | 弱い |
| WEAK | やや弱い |
| MEDIUM | 中程度 |
| STRONG | やや強い |
| VERY_STRONG | 強い |

## 余韻表示ラベル

| value | 表示名 |
|--------|--------|
| VERY_WEAK | 短い |
| WEAK | やや短い |
| MEDIUM | 中程度 |
| STRONG | やや長い |
| VERY_STRONG | 長い |
